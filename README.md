# Instructions (C0 Coverage)

Это покрытие  предоставляет информацию о количестве кода, который был выполнен или пропущен. Эта метрика полностью независима от исходного форматирования и всегда доступна, даже при отсутствии отладочной информации в файлах классов.


# Branches (C1 Coverage)

Вычисляет покрытие ветвей для всех операторов if и switch. Эта метрика подсчитывает общее количество таких ветвей в методе и определяет количество выполненных или пропущенных ветвей. Покрытие ветвей всегда доступно, даже при отсутствии отладочной информации в файлах классов. 


# Lines
Для всех файлов классов, скомпилированных с отладочной информацией, можно вычислить информацию о покрытии для отдельных строк. Исходная строка считается выполненной, если была выполнена хотя бы одна команда, назначенная этой строке.
